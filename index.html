Recommended flow

  1. Teach each concept card briefly before the matching exercise.

  3. Ask for a prediction before running exercise code.
  4. Keep exercises short and use the debriefs to lock in the idea.

Timing target

For a 60-75 minute session, keep the deck-aligned exercises in order. If the class is slower, preserve the concept explanations and shorten hands-on time by solving the last few together.
